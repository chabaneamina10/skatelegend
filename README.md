# SkateLegend

SkateLegend est un jeu de cartes inspiré du skateboard où les joueurs utilisent des tricks, combos et cartes spéciales pour battre leurs adversaires.

## Gameplay

- Construis ton deck
- Réalise des combos
- Gagne des points de style
- Utilise des cartes légendaires
- Affronte l’IA ou d’autres joueurs

## Fonctionnalités

- Système de deck
- Cartes rares et légendaires
- Tours stratégiques
- Score et multiplicateurs
- Interface moderne

## instalation

pip install pygame
